📱What is Domrey STEM?

Domrey STEM is a mobile app that makes math fun for Cambodian kids ages 3–11. It features animated Cambodian characters, Khmer voice narration, culturally relevant activities (counting mangoes at the market!), and a game-based reward system — all designed to work offline on affordable phones.

🤝Telco Partnership Model

How It Works

Domrey STEM is bundled into the telco's family/education subscription tier. Parents subscribe via their phone account — no credit card needed. Billing is handled entirely by the operator.

Subscription Flow

  • Phone-number OTP sign-in (scoped to partner operator)
  • Automatic billing via telco account
  • Subscription management in parent dashboard
  • Graceful degradation on expiry (no harsh lock-out)

Post-Exclusivity

After the locked period, the app opens to other operators or offers direct subscriptions — expanding the user base while maintaining the partnership.

📈

Value for Telco Partner

  • Unique family-friendly value-add service
  • Subscriber retention & reduced churn
  • CSR positioning in education
  • Revenue share from subscription fees
  • Brand visibility in a growing market segment
  • Data insights on family digital engagement
🛡️

TRC Alignment

  • Promotes digital literacy goals
  • Child-safe: no ads, no data collection
  • Supports national education priorities
  • Khmer-language content development

⚙️Technology Stack

📱 Mobile App

  • Flutter — cross-platform (iOS + Android)
  • Rive — interactive character animation
  • Lottie — UI animations & rewards
  • Drift (SQLite) — offline data storage
  • Firebase — auth, sync, analytics

🖥️ Backend

  • Django — admin dashboard & API
  • PostgreSQL — relational database
  • Firebase Firestore — real-time sync
  • CDN — content delivery
  • Telco API — billing integration

🎨 Animation & Audio

  • Rive State Machines — character reactions
  • Rhubarb Lip Sync — mouth animation
  • Pre-recorded Khmer — voice narration
  • AAC 64kbps — optimized audio

🤖 AI-Augmented Development

  • Claude Code — AI coding agent
  • Generates app code, tests, CI/CD pipelines
  • Enables lean team (4–5 people vs. 8–10)
  • Faster iteration cycles

Why These Choices Matter

Flutter + Rive = smooth animation on $80 Android phones. Offline-first architecture = works without internet. Data-driven content = update activities without app store releases. AI-augmented dev = ship in 6 months with a lean team.
